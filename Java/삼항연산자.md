## 삼항연산자

* if ~ else ~ 문을 간단하게 적고자 할 때 사용


만약 num의 숫자가 짝수면 even, 홀수면 odd를 result에 저장한다고 해보자.
if ~ else ~ 문을 사용하면 아래와 같이 쓰게 된다.

```java
public class Main() {
    public static void main(String[] args) {
        int num = 456;
        String result = "";
        
        if (num % 2 == 0) {
            result = "even";
        } else {
            result = "odd";
        }
    }
    
}
```

위의 코드를 삼항연산자를 사용하여 작성하면, 코드를 한 줄로 줄일 수 있다.

```java
public class Main() {
    public static void main(String[] args) {
        int num = 456;
        String result = "";
        
        result = (num % 2 == 0) ? "even" : "odd";
    }
    
}
```
